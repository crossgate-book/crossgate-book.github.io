# 🐾 賀端午祭龙祖(结束)

  • 人物等级：无
  • 必要条件：打到相应任务的 BOSS 后进入香蒂或雀儿普的房间
  • 任务奖励：护身符 or 粽子补品

# 📝 任务流程

# 1.到法兰城南区外 435.266 接取任务

老族长:端午节到了 我想去祭拜龙祖 不过山路遥远 我老人家受不了长途跋涉,年轻的勇者你可以帮我这个忙吗? 帮忙我们一族完成祭龙祖 (是 否 得到某信物)

# 2.使用传送维村 或使用 秋刀奈出小路 至维村附近 452.422

找到 龙祖山引路人 答应愿意帮忙 (是 否 得到某信物)

# 3.找到龙祖山:入口处,族人 A 指出 龙祖山 遭受不知名的邪恶力量侵蚀

使得山中魔物 开始出现异变 他们已经派出 族人 B 上山 可是怕遇到危险 恳请勇者 帮忙找寻 族人 B 并且携带 能安抚 龙祖的祭品 (是 否 得到信物)

任务分三种难度 三种难度的尾王奖励是不同的【礼包】

# 4.在随机的变异迷宫中 找到族人 B 承续任务 并获得安抚龙祖的祭品

(可能会追加 多久时间内 必须 击倒龙祖 才会触发连战的游戏模式)

(异变迷宫中 可能会有 随机异变炸弹的小 boss 模式 击倒 掉落大炸弹改图 但是 以限时活动来说 非常不想这样去构想)

# 5.到达山顶 看见 黑化的龙祖(黑化的翼龙系列 外观)

持有祭品(可能非持有也能打 就没有满足条件) 进入战斗并且打赢 二战随机出 这次感染的源头 变异大炸弹!!!（未证实） 变异大炸弹:怪物技能:自爆 同场出 1 等 大炸弹一样会自爆

对战时队长缴出：龙舟木偶 结束时缴出：祭龙祖的粽子与龙祖祭品换取礼包

王战龙祖属性为地属　建议带含有风属水晶

有【三样道具】才会触发收龙舟祭品的对话 并 出现【王弱化+大炸弹宝宝】的场合

三样 NPC 道具 会影响王战 强度
甚至有魔法封印与大地之怒等

# 6.成功解决事件 完成 老族长的要求 回到 城外南区

获得 三种难度　推测礼包为 低难度：龙祖祝福礼包　戊与己 中难度：龙祖祝福礼包　丙与丁 高难度：龙祖祝福礼包　甲与乙

任务结束

改造大炸弹材料皆由路上随机遭遇的粉红炸弹掉落，或者礼包开取 改造材料： 粉红之心　粉红火药　粉红矿石

福袋有龙祖改图 改造材料： 恶龙头颅 恶龙巨翼 恶龙之尾 暗黑基因
