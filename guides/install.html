# 🎮 游戏安装

🚥 温馨提示

安装内容为windows10系统展示，其他系统可行性未知

# 📝安装流程

# 1. 访问魔力宝贝永恒初心的官方网站 https://cg.originmood.com/ (opens new window)并于游戏下载页面内可以找到需要下载的游戏主程序和更新档，点击下载两个文件。

# 2. 找到下载的文件并解压, 文件及解压内容参考以下图片

# cg_launcher（更新档）

# cg_origin（主程序）

# 3. 双击进入cg_origin, 点击【CGSetup 1.0.0】开始安装游戏

🚨路径警告

如果安装路劲含有非法字符，如简体中文字等启动游戏之后将出现注意提示框无法登入游戏， 所以请更改为英文路径或者是繁体中文

安装时候会提供选择项，请自行勾选

  • 桌面生成快捷方式
  • 开机启动

# 4. 安装成功之后，新窗口打开更新档内文件夹CGoriginmood, 全选文件夹内所有文件, 并复制（剪切也可以但是推荐复制）

# 5. 回到刚才安装成功的魔力宝贝永恒初心的程序文件夹内, 粘贴进去, 并选择替换

# 6. 双击游戏程序即可进入游玩！ 👍 👍 👍
