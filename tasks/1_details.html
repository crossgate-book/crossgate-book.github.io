# 📜 被抹去的大陆

在阳光普照的里谢里雅堡后花园，因一句【冒险之心永不歇止】牵扯出法兰王国的古代秘辛。 随着勇者们的探索，将一步步揭开层层迷雾，深入魔兽聚集的神木，追寻禁地的真相。

猜测：此处为官方发布的任务描述，任务开始位置以及关键词的框选都表明了这个任务有关的关键词，并且表明了与法兰王国古代不为人知的机密有关，迷雾则暗示了这次任务的扑所迷离，然后神木与禁地也让大家知道了即将去追查的真相。

  • 秘辛 (opens new window)为指极其机密、隐私的事情，又或者书面指代秘密。

# 📝 任务剧情

# 里谢里雅堡后花园(30.24) 脚步蹣跚的老妇人

：

对话1： 我家老头子，天天跑图书馆，过了饭点都不知道回家！你看我腿脚又不利索的，这一来一回，菜都凉了！

对话2： 年轻人，你脚步快，能帮我去找一下那老头子吗？ 【是✅】【否❌】

对话3： 那老头一定就在里谢里雅堡里，他今天穿了一件红色马甲，应该很好找的，拜托了。

感觉这里跟剧情没啥关系...因为大家基本找到老头之后就动身去找仓库了，老奶奶工具人实锤....而且对话完老爷爷继续回来，老奶奶还是会说一样的话，估计是个bug

# 里谢里雅堡二楼左转尽头的图书馆内与看书入迷的老头子（1.74）对话

（重点人物⭐️）：

对话1： 那老太婆可真烦人！比起对未知世界的探索，吃饭算什么！

对话2： 咕噜~~~~（老头的肚子传出一阵打鼓声）

对话3： 好吧，人是铁，饭是钢，不吃饱了怎会有力量去探索新世界呢~

对话4： 少年啊，看来你对未知世界也很感兴趣，为了感谢你，我就把这段时间调查到的告诉你吧！

对话5： 你知道吗？现在我们所看到的世界地图其实并不完整。根据我祖上收藏的一本古籍记载，芙蕾雅大陆的南方还有一块宽广的大地...

对话6： 但为什么近代的地图中，那块大地会消失了呢？如果是因为天灾而沉入海底，那为什么没有任何书籍记载？

对话7： 如果是认为的刻意抹去那块大陆的相关资料，那么回事什么人能这样做？那块大地有隐藏着什么秘密呢？

对话8： 哎，都怪我老了，要不然我一定要亲自去查探！少年啊，见你人这么好。要不你代我去一趟？【是✅】【否❌】

对话9： 据我发现，要去查探那块大地的可不止我一个，而且那群人已经成立了一个组织。

对话10： 他们一直在秘密查探着，但这事可是军方禁止的，组织中已有不少人被捉走。

对话11： 剩下的人员都躲在城内某处，你可以去城南的某个仓库看看、

对话12： 记住！他们的暗号是【冒险之心永不歇止】。一切行动可都要小心，特别是不要被军方的人发现！

这里解释了这次任务我们即将面对的法兰城内的某个神秘组织，他们也在探索者芙蕾雅大陆的南方的一块被人刻意抹去的一快地区。 神秘组织藏身在城南的某个仓库，并且以【冒险之心永不歇止】为暗号来接头。

# 3. 前往法兰城内东南部的一处存储用仓库(192.177)，面对门口说出【冒险之心永不歇止】被传送入仓库内部

# 4. 与守卫（11.14）对话

对话1： 居然能找到暗道，这就不能让你离开了！

战斗对话1： 不管你是什么人！知道秘密的都得绑起来！

战斗对话2： 让我试试你的能耐！

战斗结束： 身手不错啊，应该能符合主人的要求！

这里这个守卫其实说话有点前言不搭后语...主要的信息就是进入密道的人就都得绑起来，然后身手不错的人能符合他主人的要求，也不考虑一下作为勇者的我是不是坏人，总而言之，又一颗推动剧情的垫脚石，不过比起来老奶奶算好多了，起码还有两个镜头。

然后这里追加一个脑洞，最后这句【应该能符合主人的要求】, 感觉是有被安排过来跟我们切磋切磋的。

# 5. 秘密基地内的管理人（28.26）对话

对话1： 你不是军方的人？ 看来是一场误会，真是抱歉啊，请到还有吗，让我们给你说明一下调查的结果

从这里开始可以知道这个神秘组织是对法兰军方抱有相当大的敌意，并且从愿意主动分享调查结果， 也就是对于了解这块不为人知的【禁地】的结果，是可以让更多非军方的人知晓的。

# 5. 秘密基地内的神秘组织研究员对话

对话1：墙上的这幅地图是古籍上记载的芙蕾雅大陆地图，可以看到，法兰城东南这一片已被人为损毁

对话2：从这本古籍开始，后面的所有芙蕾雅地图这篇区域都只用大海来表示。 而人们眺望的那个方向也只看到一片汪洋。

对话3：据我们多年来的观察，那片区域是被人释放了幻象魔法，在魔法的遮蔽下，任何人都无法看清魔法中的一切。

对话4：军方为何要一直隐藏那片区域呢？ 那片区域里面到底有什么秘密，这是我们组织一直探寻的目标。

对话5：我们的负责人要见你，请到里面的房间。

这里的信息就基本补充了我们的猜想，然后也解释了为何这么久以来都无人了解那块区域，基本是由于军方那边的各种阻挠。

# 6. 秘密基地内的房间，与负责人对话

对话1：又见面了~年轻人！没错，这一切是我安排（的）。我果然是没看错人，你的实力不俗啊。 加入我们吧，我们组织需要你这样的人才。 【是✅】【否❌】

对话2：很好！加入了组织你就是自己人了、

对话3：作为新人，给你一个立投名状的机会！我们组织有位重要的成员正在被羁押的司令部的地牢内。

对话4：经过连日的挖掘，我们已经挖掘出一条通往司令部地牢的地下隧道。但地牢内守卫森严，我们又无法得知成员的具体位置。

对话5：因此我们需要一位有能力的成员帮助我们前往营救。

对话6：请勿犹豫，一切已经准备就绪，我这就送你过去！

所以到这里我们才知道，其实老爷爷就是组织的负责人，然后引导我们过来的一切也都是他的安排（城里人套路深啊...）， 然后目前我们是需要通过他们挖掘的通道去营救他们的成员。

# 6. 通过隧道，来到牢房内与囚犯丁（21.8）对话

对话1：不用找了！少年，你被骗了！这里根本就没有你要找的人。和你一同过来的人趁你和守卫战斗的时候潜入上层了。

对话2：现在明白了吧，你只不过是他们的诱饵，他们的真正目标根本不是救人。军队都包围过来了，你还是乖乖束手就擒吧。

对话3：虽然已经调查清楚，你确实是被骗的，但协助罪犯盗窃军方机密的罪是跑不了了。

所以，身为勇者我们又再又再又再又再又再又再又再又再又再又再....一次被骗了，老头子压根不是为了救人，而是趁潜入的间隙盗窃军方的机密。

# 6. 传送到牢房的中央与军官（25.16）对话

对话1：念在你是异世界来的勇者，现在给你一个将功赎罪的机会。那些偷走了解除禁地结界的魔法石，现在已经把禁地的魔法解除，并潜入结界内。你是否愿意随我等进入禁地缉拿犯人 【是✅】【否❌】

对话2：那我就带你去见一见这次行动的总指挥

所以，军方念在召唤我们过来啥都没干，就愿意给我们将功赎罪的机会，然后也得知了其实老头就是为了得到解除封印的魔法石而利用了我们。

# 7. 传送至司令总部与指挥官（17.17）对话

对话1：既然你愿意加入那就必须听从我的指挥、禁地中存在某种特殊的力量，想必那些犯人是为此而去。那力量一般人是无法驾驭的，除非他是那股力量的主人又或者是来自异世界的你。因为我们才需要你的协助。

对话2：我现在给你禁地的通行证，有了这个你就可以进入禁地。请从维诺亚村南方的海底隧道前往另一座岛屿，那岛屿的东北方式前往禁地的唯一通道。到那里会有其他军方成员指引你，请务必按照指令行事。

好吧，所以这里其实也给我们说了具体的禁地所在地【请从维诺亚村南方的海底隧道前往另一座岛屿，那岛屿的东北方式前往禁地的唯一通道】

# 8. 前往禁地，然后到达底部找到犯人们

对话1：想不到你当了军方的走狗！别用这样的眼神看着我，虽然我们是利用了你，但你们人类何曾不是利用我们一族？

对话2：这力量，你们成为(错别字❌ 估计是称为)神木之心，而他正在的名字是【海洋之心】！只要它回到深海，我们一族一定能卷土从来！

对话3：拦住他，我的孩子们！！！

从这里开始其实剧情已经开始扑朔迷离了，可见老爷爷其实不是人类，而是跟神木有关系的一族，然后从他们所讲的【海洋之心】和他为什么寻求这股力量，他们很明显不是军方所说的来自异世界的我们，所以很可能就是指挥官所说的【他是那股力量的主人】，目前是什么一族还需要 我继续调查....
